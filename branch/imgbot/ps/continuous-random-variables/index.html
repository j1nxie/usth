3. Continuous Random Variables¶

3.1. Continuous Random Variables¶

(Refer to Discrete Random Variables first.)

Given the same definition, but x is an interval instead of some exact value.

3.1.1. Probability Density Function¶

A random variable X is called continuous if there is a non-negative function fX, called the probability density function of X, such that:

P(a≤X≤b)=∫abfX(x)dx

Note

The PDF has three properties:

  • fX(x)≥0 for all x

  • :math:int_{-infty}^infty f_X(x) dx = 1

  • For any subset B of the real line:

P(X∈B)=∫BfX(x)dx

3.1.2. The Mean¶

The expected value of a random variable X with PDF fX is defined by:

E[X]=∫−∞∞xfX(x)dx

Note

Expectations have linearity, as demonstrated below:

E[aX+b]=aE[X]+b

3.1.3. Expected Value Rule¶

(Refer to Expected Value Rule.)

E[g(X)]=∫−∞∞g(x)fX(x)dx

3.1.4. Variance¶

Same as its discrete variant. Refer to Variance.

3.2. Cumulative Distribution Functions¶

The CDF of a random variable X is denoted by FX and provides the probability P(X≤x).

FX(x)=P(X≤x)

For continuous random variables:

FX(x)=P(X≤x)=∫−∞xfX(t)dt

For discrete random variables (Refer to Discrete Random Variables.):

FX(x)=P(X≤x)=∑−∞xpX(k)

Note

CDF has three properties:

  • Non-decreasing

  • As x→∞, FX(x) approaches 1.

  • As x→−∞, FX(x) approaches 0.

3.3.2. Standard Gaussian Random Variable¶

A continuous random variable X is said to be standard normal N(0, 1) if it has a PDF of the form:

fX(x)=12πe−x22

3.3.3. Gaussian Random Variable¶

A continuous random variable X is said to be normal or Gaussian if it has a PDF of the form:

fX(x)=12πσe−(x−μ)22σ2

This is the most important PDF in probability theorem.

Note

The Gaussian random variable has two properties:

  • E[X]=μ

  • var[X]=σ2

3.4. Conditional PDF on an Event¶

The conditional PDF of X, given an event A with P(A)>0 is defined by:

P(X∈B|A)=∫BfX|A(x)dx

3.4.1. Conditional Expectations on an Event¶

(Refer to Conditional Expectations on an Event.)

The conditional expectation of X given an event A with P(A)>0 is defined by:

E[X|A]=∫xfX|A(x)dx

3.4.2. Expected Value Rule¶

(Refer to Expected Value Rule.)

For a function g(X):

E[g(X)|A]=∫g(x)fX|A(x)dx

3.4.3. Total Expectation Theorem¶

(Refer to Total Probability Theorem.)

If A1,…,An be disjoint events that form a partition of the sample space, with P(Ai)>0 for all i, then:

E[X]=∑i=1nP(Ai)E[X|Ai]

3.5. Multiple Continuous Random Variables¶

3.5.1. Joint PDF¶

Two random variables X and Y, associated with the same experiment are jointly continuous if they can be described with a joint PDF fX,Y:

P((X,Y)∈B)=∬(x,y)∈BfX,Y(x,y)dxdy

3.5.2. Marginal PDF¶

We can calculate the PDFs of X and Y using the formulas:

fX(x)=∫∞fX,Y(x,y)dyfY(y)=∫∞fX,Y(x,y)dx

3.5.3. Joint CDF¶

If X and Y are two random variables associated with the same experiment, their joint CDF is defined by:

FX,Y(x,y)=P(X≤x,Y≤y)

With two continuous random variables:

P(X≤x,Y≤y)=∫−∞x∫−∞yfX,Y(s,t)dsdt

3.6. Conditional PDF on a Random Variable¶

The conditional PDF fX|Y of X given Y is defined by:

fX|Y(x|y)=fX,Y(x,y)fY(y)

The conditional PDF of X given Y=y is related to the joint PDF by:

fX,Y(x,y)=fY(y)fX|Y(x|y)

3.6.1. Conditional Expectations on a Random Variable¶

The conditional expectation of a random variable X given a value y of Y is defined by:

E[X|Y=y]=∫xfX|Y(x|y)dx

3.6.2. Total Expectation Theorem¶

(Refer to Total Expectation Theorem first.)

Given two random variables X and Y:

E[X]=∫fY(y)E[X|Y=y]dy

3.6.3. Independence¶

Two random variables X and Y are independent if:

fX,Y(x,y)=fX(x)fY(y)

Note

If two random variables are independent:

  • E[XY]=E[X]E[Y]

  • var(X+Y)=var(X)+var(Y)
