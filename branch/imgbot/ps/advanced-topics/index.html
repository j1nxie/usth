4. Advanced Topics¶

4.1. Derived Distributions¶

4.1.1. Linear Function of a Discrete Random Variable¶

Given Y=aX+b:

pY(y)=pX(y−ba)

4.1.2. Linear Function of a Continuous Random Variable¶

Given Y=aX+b:

fY(y)=1|a|fX(y−ba)

4.1.3. Linear Function of a Normal Random Variable¶

Given X N(μ,σ2) and Y=aX+b:

fY(y)=12πσe−(y−ba−μ)22σ2

4.1.4. PDF of a General Function¶

Two-step procedure:

  1. Find the CDF of Y: FY(y)=P(Y≤y)

  2. Differentiate: fY(y)=dFY(y)dy

4.2. Covariance¶

The covariance of two random variables X and Y is defined by:

cov(X,Y)=E[(X−E[X])(Y−E[Y])]

When cov(X,Y)=0, X and Y are uncorrelated.

Note

Covariance has four properties:

  • cov(X,Y)=E[XY]−E[X]E[Y]

  • cov(X,Y)=var(X)

  • cov(X,aY+b)=a×cov(X,Y)

  • cov(X,Y+Z)=cov(X,Y)+cov(Y,Z)

4.3. Correlation¶

The correlation coefficient ρ(X,Y) is defined by:

ρ(X,Y)=cov(X,Y)var(X)var(Y)

The above coefficient satisfies −1≤ρ(X,Y)≤1.
