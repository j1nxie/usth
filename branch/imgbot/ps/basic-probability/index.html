1. Basic Probability¶

1.1. Probability Theory Overview¶

1.1.1. What is probability?¶

Probability theorem provide rules for systematically thinking about uncertain situations. There are several definitions:

  • Physical tendencies

  • Long-term frequency

  • Degree of beliefs

In short, these definitions complete eachother.

It could be used for predictions.

1.1.2. A probabilistic model¶

A probabilistic model is a quantitative description of an experiment whose outcome is uncertain, e.g.:

  • tossing a die

  • waiting time in a row

  • taking an exam

Putting a model together involves two key steps:

Step 1:

Describe the possible outcomes, which gives the sample space.

Step 2:

Specify a probability law, which describes the likelihood of the outcomes.

1.2. Sample Space¶

1.2.1. Definition¶

A set of all possible outcomes, notated by Ω.

All elements in the sample space must be mutually exclusive and collectively exhaustive.

Ω can be:

  • discrete: 1,2,3,…

  • continuous: [0,1]

  • finite

  • infinite

  • etc.

1.3. Probability Laws¶

1.3.1. Definition¶

Event is a subset of the sample space. Probability is assigned to events.

Probabilities have to satisfy certain basic properties to be useful, called Kolmogorov’s Axioms:

  1. Non-negativity: P(A)≥0

  2. Normalization: P(Ω)=1

  3. Additivity: If A∪B=∅, then P(A∪B)=P(A)+P(B).

Due to the above axioms, we have the following consequences:

  1. P(A)≤1

  2. P(∅)=0

  3. P(A)+P(AC)=1

  4. If A∪B∪C=∅ then P(A∪B∪C)=P(A)+P(B)+P(C)

  5. If A⊂B then P(A)≤P(B)

  6. P(A∪B)=P(A)+P(B)−P(A∩B)

1.4.2. Total Probability Theorem¶

Given a sample space Ω, partitioned into three sections: A1, A2, A3. For every i, we have P(Ai) and P(B|Ai).

From that, we can calculate P(B) by:

P(B)=∑iP(Ai)P(B|Ai)

1.5. Independence¶

An intuitive explanation of independence would be: The probability of rolling a fair die and getting a 6 while it is raining, because the result of rolling a die wouldn’t be affected by the weather.

An intuitive mathematical definition would be similar to the one below:

P(B|A)=P(B)

The formal definition of independence is:

P(A∪B)=P(A)P(B)

1.5.1. Conditional Independence¶

Given a sample space Ω, with two events A, B, knowing that event C happens.

We have the definition for conditional independence, given C:

P(A∪B|C)=P(A|C)P(B|C)
