3. Roots of Non-linear equations¶

In this section, we will go through two of the methods to find the roots of a non-linear equation: Bisection method and Newton-Raphson method.

3.1. Bisection method¶

The bisection method is a very simple method to find the roots of a non-linear equation, also called the binary search method.

Although the method is simple, it is not very efficient. It is also not very robust. The method is based on the fact that the function must have different signs at the endpoints of the interval. The method also requires that the function be continuous. If the function doesn’t have the above requirements, the bisection method will not work.

Step 1: Choose the interval [a,b] such that f(a) and f(b) have different signs.

Step 2: Find the midpoint of the interval:

c=a+b2

Step 3: If f(c)=0 then c is the root of the equation. If f(c)≠0, then repeat steps 2 and 3 with the new interval [a,c] or [c,b] depending on the sign of f(c).

  • If f(c)<0, then we assume a=c.

  • If f(c)>0, then we assume b=c.

Step 4: Repeat steps 2 and 3 until the desired accuracy is achieved.

Example: Find the root of the equation f(x)=x3−x2−3 in the interval [1,2].

First, find the midpoint of the interval, in this case:

c=1+22=1.5

Then, evaluate the function at the midpoint:

f(1.5)=(1.5)3−(1.5)2−3=−1.875

Because f(c)<0, we assume a=c. Therefore, we repeat the process with the interval [1.5, 2].

c=1.5+22=1.75

Then, evaluate the function at the midpoint:

f(1.75)=(1.75)3−(1.75)2−3=−0.7031

Because f(c)<0, we assume a=c. Therefore, we repeat the process with the interval [1.75, 2].

3.2. Newton-Raphson method¶

Newton-Raphson is a root-finding algorithm which produces successively better approximations to the roots (or zeroes) of a real-valued function. The most basic version starts with a single-variable function f(x) defined for a real variable x, the function’s derivative f′(x), and an initial guess x0 for a root of f(x).

If the function f(x) is defined and differentiable in an open interval containing a root, then the Newton–Raphson method will converge to a root of the function. The method generates a sequence of approximations xn that converges to a root of f(x) if the initial guess is close enough, and the function f(x) is suitably smooth.

Step 1: Evaluate f′(x)

Step 2: Choose an initial guess x0

Step 3: Find the next guess xi using the formula:

xi+1=xi−f(xi)f′(xi)

Step 4: Repeat steps 2 and 3 until the desired accuracy is achieved.

Example: Find the root of the equation f(x)=x3−x2−3 using the Newton-Raphson method, with initial guess x0=1.

First, evaluate the derivative of the function:

f′(x)=3x2−2x

Then, find the next guess using the formula:

x1=1−f(1)f′(1)=1−−31=4
