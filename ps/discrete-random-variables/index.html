2. Discrete Random Variables¶

2.1. Discrete Random Variables¶

The formal definition of discrete random variables is as follows, where x is some exact value.

X: sample space Ω→Routcome ω→x

Note

Notation for the random variable is X, while its numerical value is notated by x.

  • One sample space could have multiple random variables.

  • A function of random variable(s) is also a random variable.

2.2. Probability Mass Function¶

Probability mass function (PMF) is also known as the Probability Law or Probability Distribution. It is a function that gives the probability that a DRV is exactly equal to some value.

Its formal definition is given below:

pX(x)=P(X=x)

Note

Two of PMF’s basic properties are:

  • pX(x)≥0

  • ∑xpX(x)=1

2.3. Common Random Variables¶

2.3.1. Bernoulli Random Variable¶

Given a Bernoulli random variable with parameter p∈[0,1]:

X={1w.p. p0w.p. 1−p

The Bernoulli RV indicates an event A happening.

2.3.2. Uniform Random Variable¶

Given a uniform random variable with parameters a, b, and considering the following experiment: Picking one of a, a+1,… b at random, with equally likely chances.

We will have its sample space being {a,a+1,…,b}. From that, we obtain the random variable X as given below:

X(ω)=ω

2.3.3. Binomial Random Variable¶

Given a binomial random variable with parameters being positive integers n,p∈[0,1] and considering the following experiment: n independent tosses of a coin with P(Heads)=p.

We will obtain the sample space being a sequence of H and T, of length n. From that, we obtain the random variable X as given below:

X: number of heads

We have the formula of a binomial random variable as follows:

pX(k)=(nk)pk(1−p)n−k

2.3.4. Geometric Random Variable:¶

Given a geometric random variable with parameter 0<p≤1 and considering the following experiment: Infinitely independent tosses of a coin with P(Heads)=p.

We will obtain the sample space being an infinite sequence of H and T. From that, we obtain the random variable X as given below:

X:number of tosses until the first head

We have the formula of a geometric random variable as follows:

pX(k)=(1−p)k−1p

2.4. Expectations¶

2.4.1. Mean¶

The expected value (also called the expectation or the mean) of a random variable X, with PMF pX is defined by:

E[X]=∑xxpX(x)

Expectations have linearity, demonstrated below:

E[X]=aE[X]+b

2.4.2. Expected Value Rule¶

Let X be a random variable, and Y=g(X) be a function of X. By definition:

E[g(X)]=E[Y]=∑yypY(y)

From that, we can obtain the general form of the expected value rule for any function g(X) of random variable X:

E[g(X)]=∑xg(x)pX(x)

2.4.3. Variance¶

The variance is the average distance from the mean, or in other words, the expected value of the random variable g(X)=(X−E[X])2.

var(X)=E[(X−E[X])]2

With the variance, we can also obtain the standard deviation:

σX=var(X)

Note

Variance has two properties, given a random variable X:

  • var[aX+b]=a2var[X]

  • var[X]=E[X2]−(E[X])2

2.5. Conditional PMF on an Event¶

2.5.1. Definition¶

The conditional PMF of X, given an event A with P(A)>0 is defined by:

pX|A(x)=P(X=x|a)=P({X=x}∪A)P(A)

Note

Sanity check: verify that:

∑xpX|A(x)=1

2.5.2. Conditional Expectations on an Event¶

The conditional expectation of X given an event A with P(A)>0 is defined by:

E[X|A]=∑xxpX|A(x)

Note

This is very similar to the normal expectation of a discrete random variable. (Refer to Mean.)

From that, we also have the expected value rule:

E[g(X)|A]=∑xg(x)pX|A(x)

2.5.3. Total Expectation Theorem¶

(Refer to Total Probability Theorem first.)

If A1,…,An be disjoint events that form a partition of the sample space, with P(Ai)>0 for all i, then:

E[X]=∑i=1nP(Ai)E[X|Ai]

2.6. Multiple Random Variables¶

2.6.1. Joint PMF¶

Consider two discrete random variables X and Y, associated with the same experiment (very important!). The joint PMF of X and Y is defined by:

pX,Y(x,y)=P({X=x}∪{Y=y})=P(X=x,Y=y)

2.6.2. Marginal PMF¶

We can calculate the PMFs of X and Y seperately (referred to as the marginal PMF of X and Y) by using the formulas:

pX(x)=∑ypX,Y(x,y)pY(y)=∑xpX,Y(x,y)

Note

Proof of the above formula:

pX(x)=P(X=x)=∑yP(X=x,Y=y)=∑ypX,Y(x,y)

2.6.3. Expected Value Rule¶

A function g(X,Y) of X and Y defines another random variable, and:

E[g(X,Y)]=∑x∑yg(x,y)pX,Y(x,y)

Note

The above expected value rule has linearity:

E[aX1+bX2+c]=aE[X1]+bE[X2]+c

2.7. Conditional PMF on a Random Variable¶

2.7.1. Definition¶

The conditional PMF P(X=x|Y=y) of X given Y is defined by:

pX|Y(x|y)=P({X=x,Y=y})P(Y=y)=pX,Y(x,y)pY(y)

Note

Sanity check: verify that:

∑xpX|Y(x|y)=1

The conditional PMF of X given Y=y is related to the joint PMF by:

pX,Y(x,y)=pY(y)pX,Y(x|y)

2.7.2. Conditional Expectations on a Random Variable¶

The conditional expectation of a random variable X given a value y of Y is given by:

E[X|Y=y]=∑xxpX|Y(x|y)

2.7.3. Total Expectation Theorem¶

Given two random variables X and Y, the unconditional average can be obtained by averaging the conditional averages.

E[X]=∑ypY(y)E[X|Y=y]

2.7.4. Independence¶

Two random variables X and Y are independent if:

pX,Y(x,y)=pX(x)pY(y)

Note

This is similar to basic probability’s independence. (Refer to Independence.)

Some properties:

  • E[XY]=E[X]E[Y]

  • var(X+Y)=var(X)+var(Y)
